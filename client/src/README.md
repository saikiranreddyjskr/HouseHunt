# HouseHunt

HouseHunt is a full-stack house rental web application that allows users to search, view, and manage rental properties through a modern and user-friendly interface.

## Features
- User Registration and Login
- Browse Available Properties
- Search and Filter Houses
- Property Details Page
- Responsive Design
- Database Integration

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Project Structure
- Frontend
- Backend
- Database
- Documentation
